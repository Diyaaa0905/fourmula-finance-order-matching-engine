Problem Statement: Design and implement a stock exchange order matching engine that processes buy and sell orders from multiple traders

This project implements a stock exchange order matching system that simulates how trading platforms process buy and sell orders. Traders submit orders containing details such as price, quantity, and stock ticker. The system first validates each order using market rules like tick size, available balance, and user holdings. Valid orders are placed into a queue and then processed by the exchange, which routes them to the appropriate stock’s order book. The system is designed to handle multiple stocks, maintain user profiles, and ensure fair and efficient order handling, forming the foundation for trade matching and execution.
